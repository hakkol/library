## Installation

Run:
```
git clone https://github.com/hakkol/library.git
```
to clone the repository.

Run:

```
cd projectname
composer install
```


Copy ***.env.example*** to ***.env***.

Run:

```
php artisan key:generate
```
to regenerate secure key.

Run:

```
php artisan migrate
```
to create and populate tables.

Run:

```
npm install
npm run dev
```
to manage assets.

Run:
```
php artisan scraper
```

to use the scraper
